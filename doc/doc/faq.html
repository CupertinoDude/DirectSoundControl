 Frequently asked questions :

Q: Doom 3 doesn't allow me to enable Surround sound, can DSC help me ?

A: Follow these steps :
1.) Download the latest version of DirectSound Control .
2.) Unpack the content of the ZIP file to the Doom 3 directory (NOT in the windows/system(32) !!!!!!!!!!). This is where the doom3.exe is located.
3.) Open the DSoundCtrl.ini file from game directory with notepad.
4.) Find the line "ForceSpeakerConfig=0" and change the value from 0 to 1
5.) Find the line "SpeakerConfig" and change the value to 6
6.) Save the INI File.
8.) Start the game.


Q: My favorite game XXX doesn't support 3D positional audio, can DSC help me ?
A: If the game is developed with 3D sound in mind but the 3D sound is disabled for some reason try this :
1.) Download the latest version of DirectSound Control .
2.) Unpack the content of the ZIP file to the game directory (NOT in the windows/system(32) !!!!!!!!!!)
3.) Open the DSoundCtrl.ini file from game directory with notepad.
4.) Find the line "ForceVoiceManagement=0" and change the value from 0 to 1
5.) Find the line "ForceNonStaticBuffers=0" and change the value from 0 to 1
6.) Save the INI File.
8.) Start the game.


Q: How to get 3D positional audio with DeusEx - Invisible War ?

A: Follow these steps :
1.) Download the latest version of DirectSound Control .
2.) Unpack the content of the ZIP file to the DX:IW/system directory (NOT in the windows/system(32) !!!!!!!!!!)
3.) Open the DSoundCtrl.ini file from the DX:IW/system directory with notepad.
4.) Find the line "ForceHardwareMixing=0" and change the value from 0 to 1
5.) Save the INI File.
6.) Edit the properties of the DX2 shortcut on the desktop, add the DX:IW/system directory to the "Start in" field.
7.) Start the game, it should work now !!!

Q: How to get 3D positional audio with Mafia ?

A: Follow these steps :
1.) Download the latest version of DirectSound Control .
2.) Unpack the content of the ZIP file to the Mafia directory (NOT in the windows/system(32) !!!!!!!!!!)
3.) Open the DSoundCtrl.ini file from the Mafia directory with notepad.
4.) Find the line "PreventSpeakerSetup=0" and change the value from 0 to 1
5.) Save the INI File.
6.) Check that the Windows Speaker Setup  in the control panel  is set to your correct speaker configuration.
7.) Start the game, it should work now !!!


Q: I'm using the onboard sound of my nVidia nForce 1/2 board and I've got problems with your DLL.
A: It seems that the nForce chipset has problems with hardware mixed buffers. Please follow these rules :
    • DON'T enable voice management
    • DON'T force hardware mixing with this chipset.
    • Try "ForceSoftwareMixing=1" if you have problems with games, this might help
Q: The DSC DLL won't be loaded, I'm using Win9x/ME. Why  ?
A: Try to restart your computer and don't start another application beforce you start the application that should be uses with DSC. I don't recommend and support Win9x/ME with DSC.

Q: I see a major speed drop when I'm using the DSC dll. Why ?
A: You're using the developer version of DSC with logging enabled. Download the public version, overwrite the DSOUND.DLL and try again.
