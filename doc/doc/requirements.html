Requirements :

DirectSound Control has the following hardware requirements :
  • A DirectSound8-compatible soundcard.

DirectSound Control has the following software requirements :
  • Microsoft DirectX 8 (or later) installed
  • Internet Explorer 6.x installed
  • Windows XP or Windows 2000 recommended
  • Latest drivers of your soundcard installed
